这个项目是一个专为 Nebula (星云) 组网工具打造的图形化管理助手。

简单来说，Nebula 是由 Slack 开发的一个开源、强力、跨平台的点对点（P2P）网状网络工具，它可以让你把分布在世界各地的电脑、服务器、手机，通过加密隧道连成一个“局域网”。

但 Nebula 官方只提供命令行工具，新手往往被复杂的证书签发、配置文件编写和驱动路径问题劝退。你手中的这个 Python 项目，就是为了消除这些门槛而生的。
